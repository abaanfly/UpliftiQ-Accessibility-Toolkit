# Accessibility Mission

UpliftiQ Accessibility Toolkit exists to support educational products where meaning is visible, paced, and inclusive by design.

## Principles

1. **Captions are primary content**
   - Captions should be large, readable, and visually separated from decorative UI.
2. **Audio alternatives must be visible**
   - Alerts, timers, and state changes should never depend only on sound cues.
3. **Sign-friendly video matters**
   - Sign language learning needs slow playback, looping, and stable framing.
4. **Keyboard support is essential**
   - Learners and educators should be able to control the interface without a mouse.
5. **Clarity over decoration**
   - High contrast, consistent spacing, and predictable interactions build trust.

## Design outcome

The toolkit favors strong contrast, clear focus indicators, structured transcript reading, and visual reinforcement for every important interaction.
